HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element is a parent to its child elements such as li and ul.

CSS

  • Margin is the space outside an element. If set to zero there is no extra space.
  • Padding adds space around content inside an element.
  • * is a special wild card symbol(applies rule to all elements).
  • DRY (do not repeat yourself)
  • Link creates the connection to the external file.
  • rel specifies the relationship between the current document and the linked document resource.
  • href specifies the location (URL) of the external resource.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed.
  • A function is a set of instructions that tells the computer how to perform a certain task.
